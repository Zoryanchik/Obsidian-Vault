![[Pasted image 20251113034608.png]]Here's what each variable means:

- **$s$ (sequential):** The proportion of the program that **must** run sequentially (cannot be parallelized).
    
- **$1-s$ (parallel):** The proportion of the program that **can** be parallelized.
    
- **$p$ (processors):** The number of processors (or machines) you are using to run the parallel part.
