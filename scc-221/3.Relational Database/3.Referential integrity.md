## 4) “Print the orders of Customers whose Customer number starts with A?”

Two reasonable interpretations:

**A — Return only rows from `Orders` where the `Customer#` begins with `'A'`:**

`SELECT * FROM Orders WHERE Customer# LIKE 'A%';`

**B — Return order details plus customer name and item description (more useful):**

`SELECT o.Order#, o.Item#, i.Description, o.Customer#, c.Name, o.Delivery_date, o.Quantity FROM Orders o JOIN Customers c ON o.Customer# = c.Customer# JOIN Items i     ON o.Item#     = i.Item# WHERE o.Customer# LIKE 'A%';`
