![[Pasted image 20251007155500.png]]### **2️⃣ Key concepts**

**`e.key`** → the key the user pressed (`"s"` for stop, `"g"` for go)  
**`classList.remove()`** → removes a CSS class from the light  
**`classList.add()`** → adds a CSS class to the light

We want:

|Key pressed|Light color|Class added|Class removed|
|---|---|---|---|
|`"s"`|red|`"stop"`|`"go"`|
|`"g"`|green|`"go"`|`"stop"`|
### **1️⃣ Основная идея**

У нас есть элемент с двумя классами:

- `"stop"` → красный
    
- `"go"` → зелёный
    

**Логика:**

1. Если нажата **S (stop)**:
    
    - Всегда **удаляем `"go"`**, потому что мы хотим красный свет
        
    - Всегда **добавляем `"stop"`**, чтобы включить красный
        
2. Если нажата **G (go)**:
    
    - Всегда **удаляем `"stop"`**, потому что мы хотим зелёный свет
        
    - Всегда **добавляем `"go"`**, чтобы включить зелёный

## **2️⃣ Разные типы селекторов**

### **a) По ID (`#`)**

- Если у элемента есть `id="myId"`, используем `#myId`:
    

`<div id="header">Hello</div>`

`const header = document.querySelector("#header");`

- **Почему `#`?**  
    В CSS `#` означает «ID». `querySelector` принимает CSS-синтаксис, поэтому тоже нужен `#`.
    
- Вернёт **первый элемент с этим ID**.
    

---

### **b) По классу (`.`)**

- Если элемент имеет `class="heading"`, используем `.heading`:
    

`<h1 class="heading">Title</h1>`

`const heading = document.querySelector(".heading");`

- **Почему `.`?**  
    В CSS точка (`.`) обозначает класс.
    
- Вернёт **первый элемент с этим классом**.
    

---

### **c) По тегу (имя без `#` или `.`)**

- Если хотим найти элемент по тегу (например `<div>` или `<p>`), пишем просто имя:
    

`<div>Content</div>`

`const div = document.querySelector("div");`
