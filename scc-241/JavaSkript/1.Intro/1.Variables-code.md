Key differences between the three keywords relate to:  
_2.2.1 Immutability:_ const declarations are immutable for primitives and object references. var/let aren't.
```

_2.2.2 Initialization:_ var/let don't need initialization at time of declaration. Their value is 'undefined' until assigned value. const needs values for initialization.  
_2.2.3 Scope:_ var has function/global scope (lookup 'hoisting'), let/const have block level scope.
```
