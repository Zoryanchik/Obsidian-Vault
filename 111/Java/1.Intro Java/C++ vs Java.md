- **`public class HelloWorld`** – Defines a class named `HelloWorld`. In Java, all code must be inside a class.
- **`public static void main(String[] arguments)`** – This is the main method where execution begins.
    - `public`: Makes the method accessible from anywhere.
    - `static`: Allows the method to be run without creating an instance of the class.
    - `void`: Indicates that the method does not return any value.
    - `String[] arguments`: Allows command-line arguments to be passed.
- **`System.out.println("Hello world");`** – Prints `"Hello world"` to the console.
